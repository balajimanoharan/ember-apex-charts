ember-apex-charts
==============================================================================

An Ember Wrapper for [apexcharts](https://apexcharts.com)


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-apex-charts
```
